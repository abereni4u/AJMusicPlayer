Design Thoughts

Design Map (as of 11.26.2025)

Scenarios

Launching Music Player First Time

  1. Open the program for the first time and link my local music directory to the program
    • Look at default config folder + load config settings
    • Need to generate a default config file with the user's directory to create the MusicLibrary
    • Creating the MusicLibrary -Scan the given directory for music files -Create MusicLibrary object
      • Write to config folder -Create a Music object for each item and add to MusicLibrary object
      • Title
      • Artist
      • Duration
      • Cover Art
      • File Path
  2. See entire music library in viewport
    • After Music items have been added to MusicLibrary
      • Read from MusicLibrary and add Music objects to viewport
  3. I click on a song and play it.
    • Queue object is created. Based on view mode (Playlist or Library) Music objects are added to Queue
    • Clicked song is added to Play Bar object and played.
    • Play Bar methods activated
      • Play
      • Pause
      • Prev
      • Next
      • Shuffle (the Queue)
      • Repeat
      • View Queue
        • Rearrange Music in Queue
        • Add Music to Queue
        • Remove Music from Queue
        • Clear Queue
      • Be able to use keyboard shortcuts for the methods above
  4. From the viewport, I create a new playlist from one of the songs in my library.
    • Show previously made Playlist objects
    • Show create Playlist option
      • Create new Playlist object to hold Music objects
      • Copy current selected Music object's path to new Playlist object array
      • Update viewport to contain Playlist object
    • Write file to config folder
      • Only write again when program closes and if changes have been made
  5. Sort my music library
    • The viewport array of Music objects will be modified here
    • Sort by whatever value here (ratings, title, duration)
    • Same as above but for grouping (by playlist, album, etc)
  6. Close the program
    • Update config file / folder based on any changes made (to settings, new playlists, directories added, etc)
